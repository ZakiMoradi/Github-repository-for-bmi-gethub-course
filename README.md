# Calculator App

## Overview

Calculator App is a simple and user-friendly mobile application created to help users perform mathematical calculations quickly and easily. The application is designed with a clean interface so that everyone can use it without difficulty.

This project was developed as a practice project for learning mobile application development and GitHub project management.

---

## Features

- Basic mathematical operations
  - Addition
  - Subtraction
  - Multiplication
  - Division

- Simple and clean user interface
- Fast calculation results
- Responsive design
- Easy navigation

---

## Technologies Used

This application was developed using:

- Flutter
- Dart
- Material Design

---

## Purpose of the Project

The main purpose of this project is to create a functional calculator application while practicing software development skills such as:

- UI design
- State management
- User interaction
- GitHub repository management
- Documentation writing
